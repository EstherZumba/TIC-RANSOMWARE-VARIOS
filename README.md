# TIC-RANSOMWARE-VARIOS

Este repositorio contiene el desarrollo del prototipo funcional para la detección de ransomware, realizado como parte del Trabajo de Integración Curricular.

## Acceso al sistema web
El sistema web se encuentra desplegado y puede ser accedido a través del siguiente enlace:
https://tic-ransomware-frontend.vercel.app/

## Pruebas del sistema
Las pruebas realizadas sobre el prototipo, junto con los archivos utilizados para la validación, se encuentran disponibles en el siguiente enlace:
https://github.com/EstherZumba/TIC-Ransomware/tree/main/Test

Para realizar las pruebas, el usuario debe acceder al sistema web y utilizar los archivos disponibles en la carpeta de pruebas.
